A Burping The Corpse badge widget for use in your WordPress sidebar. There are 2 badge images to choose from, or no badge at all.

== Description ==

Adds a Burping The Corpse badge widget for use in your WordPress sidebar. The widget displays a Burping The Corpse badge and the latest posts from the Burping The Corpse blog (the feed and title can be customized through the widget options). There are 2 badge images to choose from, or no image.
Also, you have the option of adding an rss feed of your choice, and adding html before and after the feed, essentially allowing you to use it for any rss feed. 
Tested on WordPress 2.3 - WordPress 3.3.1.

== Installation ==

1. Download, unzip and upload the `burping-the-corpse-sidebar-widget` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. Enable the widget (`Appearance > Widgets`) and customize the options.

== Frequently Asked Questions ==

= How can I change the image displayed by the widget? =
Simply open the widget option page (`Appearance > Widgets`) and select one from the dropdown list, or select none, and add one via a bit of html in the widget options.

= This thing is messing up my sidebar! How can I fix it? =
While the widget should work with most themes out-of-the-box, you can customize the way it is displayed easily using CSS or by simply adding some HTML or inline styles using the widget options page (see the screenshots).
